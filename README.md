# Student Management System (Java)

A console-based Java application to manage students.

## Features
- Add student
- View all students
- Delete student by ID
- Search student by ID
- Update student name
- Student count
- Grade tracking
- Duplicate ID validation

## Technologies
- Java
- OOP concepts
- ArrayList
- Scanner

## How to Run
1. Open project in IntelliJ IDEA
2. Run Main.java
3. Use the menu to manage students
