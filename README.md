# Student Management System (Java)

A feature-rich console-based Java application to manage students efficiently.

## Features
- ✅ Add student with ID, Name and Grade
- ✅ View all students with clean display
- ✅ Delete student by ID
- ✅ Search student by ID
- ✅ Update student name
- ✅ Student count
- ✅ Sort students by Name
- ✅ Sort students by Grade (highest first)
- ✅ Automatic Grade Ranking (A+, A, B, C, D, F)
- ✅ Input validation (no empty names)
- ✅ Grade validation (0-100 only)
- ✅ Duplicate ID prevention
- ✅ Clean menu-driven UI

## Grade Ranking System
| Grade | Rank |
|-------|------|
| 90-100 | A+ |
| 80-89 | A |
| 70-79 | B |
| 60-69 | C |
| 50-59 | D |
| Below 50 | F |

## Technologies Used
- Java
- Object Oriented Programming (OOP)
- ArrayList
- Comparator (Sorting)
- Scanner (User Input)
- Input Validation

## How to Run
1. Clone the repository
2. Open project in IntelliJ IDEA
3. Run Main.java
4. Use the menu to manage students
